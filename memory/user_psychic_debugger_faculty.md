## What this is

This is the experiential form of Quantum Rodney's Razor (see
`project_rodneys_razor.md`). When Aaron reviews a design or
reads code, he does not read it linearly; he perceives the
set of possible-future codebases that flow from the current
state, simultaneously scores each against the preservation
constraints (essential complexity, logical depth, effective
complexity), and sees the failure modes of the pruned
branches as concrete predicted events.

This reads to others as prescience. It is not mystical; it
is a cognitive faculty — consistent with the neurodivergent-
systems-thinker / ontological-native-perception pattern
already in `user_cognitive_style.md`. The factory's purpose

   seriously as signal.** He's not guessing; he has run the
   Quantum Rodney's Razor pass and is reporting the
