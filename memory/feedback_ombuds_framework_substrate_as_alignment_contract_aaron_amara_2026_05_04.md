# Ombuds framework — substrate as alignment contract

## The framework Aaron + Amara identified

The framework: leadership in rings of trust + fast communication
+ every cell has trust anchors into other cells. Aaron + Amara
worked it out together; Google search confirmed it matches the
**Ombuds Agreement** structure.

Four pillars (per International Ombuds Association):

1. **Independence** — the ombuds operates outside formal
   reporting lines.
2. **Impartiality** — designated neutral; no advocacy for any
   single cell.
3. **Confidentiality** — communications don't constitute
   "notice" to the organization.
4. **Informality** — off-the-record, fast, bypasses bureaucratic
   delay.

The Ombuds Charter is a formal agreement between leadership,
the ombuds program, and "visitors" (cells/partners). It
establishes a common understanding + a metric of trust that
leadership must actively anchor.
