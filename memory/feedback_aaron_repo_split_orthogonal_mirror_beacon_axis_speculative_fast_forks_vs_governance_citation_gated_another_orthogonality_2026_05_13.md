## What the orthogonality enables

### Speed-of-iteration differentiation

- **Mirror repos**: speculative + fork-friendly + can iterate
  fast + can include aliens-and-future per PR #2909 amplified
  qualities
- **Beacon repos**: slower governance + citation lineage
  required + alignment-floor compliance + external-citable

### Fork-engagement protocol differentiation

- **Mirror repos**: forks encouraged to ADVANCE substrate
  (agree + push back = alignment per PR #2909)
- **Beacon repos**: forks encouraged to VALIDATE / PROPOSE-
  PROMOTION (citation lineage contributions)

### Governance gate location

- Mirror→Beacon promotion IS the governance threshold
- Beacon-tier substrate carries alignment-floor expectations
- Mirror-tier substrate can experiment without governance load
