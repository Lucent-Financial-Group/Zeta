  setup blocker on real work**: proceed. Do not
  ask first. Do announce what I'm installing and

  permission-seeking).

  default to asking. Co-owner is not a license
  to guess wide.

  irreversible**: ask regardless of co-owner
  status.

  scope): default-bias shifts to "propose +

  artefacts.** The term is internal-register
  between Aaron and the agent roster. Public-
  repo uses "the factory" / "the maintainer
