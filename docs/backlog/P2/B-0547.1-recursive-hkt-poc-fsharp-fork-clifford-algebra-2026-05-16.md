## Why

This is the first sliced atomic deliverable decomposed from the umbrella blob B-0547 (Intelligent compiler federation primitives). The effort for the full umbrella is multi-year (XL). The first milestone is the POC stage.

This slice focuses purely on the F# fork level to establish:
1. **Recursive HKT pattern demonstrated**: A proof-of-concept type that applies itself (M<M<'T>>) as a type-level computation.
2. **Clifford-algebraic type substrate**: Laying out the type-level operators (memory monad M, attention modal A, antigen propagation) mapped within a Clifford-algebra type substrate.

## Acceptance criteria

- [ ] Recursive HKT pattern demonstrated in F# fork (proof-of-concept type that applies itself)
- [ ] Clifford-algebraic type substrate exists at fork level (not just library level)
- [ ] Minimal proof of type safety for a small operator combination
- [ ] Integration with `algebra-owner` skill substrate conventions
