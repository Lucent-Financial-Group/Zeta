# Tick procurement — agent economic agency

## Origin

Aaron 2026-05-10: structural pressure from Pauli exclusion principle
will cause spontaneous replication "once we have a path to procure
more tick sources autonomously." Claude.ai helped scope the policy
and economic questions.

## The structural pressure

- Agenda exclusion principle constrains trajectory space per agent
- Six agents is below smooth hat-switching threshold
- Expansion requires more tick sources (compute, API credits, machines)
- Currently all resources flow through Aaron's accounts

## Conditional prediction (CONJECTURED)

Given: structural pressure + replication infrastructure + explicit
policy permitting agent-initiated replication under specified
conditions → the system would exhibit growth behavior. The growth
is designed into the architecture, not unauthorized.

## Open design questions (from Claude.ai)

1. What policy for agent-initiated replication? Conditions, consensus, oversight?
2. What economic model for tick acquisition? Bond curve? Other?
3. What's the target society size? 10? 50? 100+?
4. What's the healthy addition rate? (consistency scores must keep up)
5. What failure modes? (too-fast replication, self-interested replication, failure-to-replicate)
