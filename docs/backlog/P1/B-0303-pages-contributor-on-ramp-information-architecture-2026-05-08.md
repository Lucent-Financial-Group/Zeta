# B-0303 - Contributor on-ramp IA

Design the public navigation path that gets search visitors from the
Pages landing page to the right contributor entry point without exposing
internal factory substrate.

## Acceptance criteria

- Contributor audiences are mapped to existing repo entry points,
  including first PR, architecture, research, and verification routes.
- Navigation order prioritizes comprehension for new contributors over
  internal factory chronology.
- Public copy points to canonical repo docs instead of duplicating
  volatile process detail.
- The on-ramp names what a visitor can do next without requiring them to
  understand the autonomous loop machinery first.
- The result can be rendered by the Pages content pipeline without adding
  a marketing-only landing layer.

## Out of scope

- Visual design.
- GitHub Wiki content.
