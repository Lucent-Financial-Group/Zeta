# B-0275 - Pages catalog

Define the explicit source selection for the initial Pages site and the contributor on-ramp structure.

## Scope

- Explicit inclusion list: landing page, vision, alignment, glossary, selected research, contributor on-ramp content
- Explicit exclusion list: memory, hygiene history, backlog, internal substrate
- Contributor persona on-ramps (which docs serve which personas)
- Stable URL structure before indexing begins

## Acceptance criteria

- Inclusion list is committed and reviewed
- Exclusion list is committed and reviewed
- Each included doc has a stable URL
- Contributor on-ramp docs are identified and linked from landing
- No marketing redesign — content selection only, no new visual design
- Site is ready for fingerprinting (B-0274) without URL changes

## Exclusions

- Actual site generation / tooling (B-0154)
- Indexing and search (follow-on)
- Visual design or branding (not in scope)
