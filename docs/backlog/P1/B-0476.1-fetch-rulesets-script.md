# Script to fetch and dump rulesets from all repos

## Purpose
This is a decomposed slice of B-0476. We need a TypeScript script that automates the `gh api` calls to fetch rulesets across multiple repos (LFG/Zeta, AceHack/Zeta, LFG/civsim) and outputs them as structured JSON so the divergence matrix can be generated.

## Requirements
- Author `tools/research/fetch-github-rulesets.ts`.
- Wrap `gh api repos/<org>/<repo>/rulesets`.
- Extract ID, name, target, and enforcement levels.
- Output clean JSON.
