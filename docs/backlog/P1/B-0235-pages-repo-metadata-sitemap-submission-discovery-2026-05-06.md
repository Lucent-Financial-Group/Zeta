# B-0235 - Repo metadata and discovery signals

Connect the live Pages surface to GitHub and external search
discovery channels.

## Work scope

This row owns repository topics, repository homepage,
description review, sitemap submission, and the measurable
signals that show the discovery funnel is alive.

## Acceptance criteria

- Repository topics include the honest technical discovery
  terms from B-0154.
- Repository homepage points at the live Pages URL.
- Sitemap submission steps for Google Search Console and Bing
  Webmaster Tools are documented or completed.
- Discovery signals are defined: time-to-index, target-query
  ranking, crawler hit rate, and external contributor source.
