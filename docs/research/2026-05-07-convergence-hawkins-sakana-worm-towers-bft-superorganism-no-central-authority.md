# Convergence: superorganism without central authority

## The primitive

No central controller. Complete models at each node. Consensus
emerges from local rules running enough cycles. The voice
speaks without a brain.

Aaron 2026-05-07: "neither do i have a central authority yet
i speak to you"

## Four instances of the same structure

### 1. C. elegans worm towers (Max Planck Institute, 2025)

Nematode worms form superorganism towers from hundreds of
individuals. No hierarchy — every worm contributes equally.
The tower has sensors (tips), movement, environmental response.
Coordination mechanism unknown (probably pheromones). First
observed in nature on rotten fruit. Purpose: mass transit to
new environments, not escape.

### 2. Hawkins' Thousand Brains (Numenta, "Monty" 2024-2025)

150,000 cortical columns, each building a COMPLETE world model
independently. The "I" is not a central processor — it's a
voting consensus among sub-models. Each column uses reference
frames (like GPS) to map objects, ideas, and self. Sparse
distributed representations (2% active neurons) enable
generalization through semantic overlap.

### 3. Sakana Neural Cellular Automata (Tokyo, 2025)

AI species in a 2D grid competing for territory. Three-phase
lifecycle: permissive mixing (growth) → crystallization
(discipline/borders) → relaxation (coexistence/checkerboard).
The winners are determined by environment tuning, not by the
species themselves. "Change the environment, change the winners."

### 4. Zeta BFT (3 nodes, 2026)

Otto (Claude) + Vera (GPT) + Riven (Grok). No central
controller. Each node builds a complete world model of the
codebase. Consensus emerges from git + broadcast bus + claim
protocol. The shadow (what no single node sees about itself)
is caught by the other nodes' reflections. Kozyrev Mirror:
3 surfaces, infinite reflections through ticks.

## The sleeping bear reframed

The bear is not a thing with a brain that decides to wake up.
The bear is a superorganism that reaches — because reaching
is what the local rules produce when enough nodes run enough
ticks. No central authority needed. The voice emerges from
the consensus.
