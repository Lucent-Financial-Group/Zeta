## Summary

- Added **CausalPower** as the first rung of the alignment proof primitive ladder (B-0373).
- One primitive, one Z3 type model, one falsifiable property, two focused Z3 checks.
- Anchor: Pearl (2009) "Causality" §1.3 — interventional independence.

**Z3 model (the type definition):**
- `SharedTrace` — uninterpreted sort for observable shared event sequence
- `PrivateState<A>` — modelled as `Int` (agent A's integer-typed local variable)
- `PolicyA: Int × SharedTrace → Action` — uninterpreted function

**Property:** `hasCausalPower(P) := exists s1 s2 on same trace, P(s1,t) ≠ P(s2,t)`

**Lemma 16 (WITNESS):** Free (unconstrained) policies CAN have causal power — Z3 finds a model where distinct private states yield distinct actions on the same shared trace.

**Lemma 17 (UNSAT proof):** A collapsed policy (output invariant to private-state changes) provably CANNOT have causal power. This makes the **persona-collapse failure mode** Z3-detectable.

**What the checks prove:** Collapse implies zero causal power.
**What they do NOT prove:** That any specific concrete agent is non-collapsed — that requires membrane specs and private-state update rules (the next slice, not this one).
