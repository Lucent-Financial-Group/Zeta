### B-0662 — Closed bidirectional causal loop chain

> *"Spec ↔ F# ↔ C# ↔ Rust. Each pair has to be able to regenerate the other. The moment that bidirectional link breaks, you know something has gone wrong... like having checksums at every layer, except the checksum can fully reconstruct the thing it's checking."*

**Sharpens B-0632** (no-privileged-implementation; mutual regeneration) into concrete chain-of-pairs structure. Each adjacent pair must support bidirectional regeneration; broken link = integrity-violation signal.

Composes with B-0649 permanent coliseum (competition criterion = bidirectional-link-survival) + B-0652 three-faction BFT (TLA+ at Spec layer) + B-0653 persistent integrator (round-trip success = health metric).
