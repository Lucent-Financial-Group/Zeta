## Summary

Slice 2 of B-0440. The Standing-by detector now does **real detection** by polling \`git log\` for the most recent commit on HEAD and comparing the timestamp against \`idleThresholdMin\`.

## What changed from slice 1

- spawnSync (execFile-style, no shell) — no command-injection risk
- Adapter pattern (\`now\` + \`lastCommitIso\`) — tests inject deterministic values
- Idle threshold is INCLUSIVE (\`>=\`); boundary flags idle
- Clock-skew safe: negative idleMinutes clamp to 0
- Handles missing git history (fresh repo / git unavailable) without crashing
- Result schema extended: \`lastCommitAt\`, \`idleMinutes\`, \`idleDetected\`, \`note\`
