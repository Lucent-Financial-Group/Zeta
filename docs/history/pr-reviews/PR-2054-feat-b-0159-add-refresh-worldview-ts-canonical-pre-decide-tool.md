## Description

## Summary

- Adds `tools/github/refresh-worldview.ts` -- a single TS script that gathers open PRs, recently merged PRs, open issues, and local git state into one structured JSON snapshot
- Follows the same `spawnSync` + structured-JSON pattern as `poll-pr-gate.ts` and `poll-pr-gate-batch.ts`
- Supports `--owner`, `--repo`, `--pr-limit`, `--merge-limit`, `--issue-limit` flags with sensible defaults (LFG/Zeta, 50/5/20)
- Replaces ad-hoc `gh` + `git` command chains for the refresh-before-decide invariant (CLAUDE.md)

Closes B-0159 scope (canonical refresh tool). Composes with B-0262/263/264.
