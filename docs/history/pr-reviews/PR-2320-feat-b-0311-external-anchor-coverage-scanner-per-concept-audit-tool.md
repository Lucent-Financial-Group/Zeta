## Description

## Summary

- Adds `tools/alignment/audit_external_anchors.ts`: given the concept registry (B-0310), locates each concept in its source surface and extracts external URLs from a ±20-line context window.
- Classifies each URL as `paper` / `rfc` / `blog` / `so-se` / `talk` / `other` by domain matching.
- Emits a JSON or Markdown coverage report mapping each concept to `anchored` (URL list) or `anchor-pending`.
- CLI flags: `--json`, `--md`, `--out DIR`; exit codes 0 / 2.
