## Design rationale

- **Infrastructure before translation**: B-0004.3 (drift lint) and B-0004.4 (xref validator) are required gates before *any* translated file ships. Without them, translations would silently diverge from English source and break cross-references for monolingual readers.
- **ADRs before pilots**: B-0004.5 (pipeline ADR) documents the AI provider choice with quality metrics; B-0004.6 documents the language choice with community outreach. Both decisions are retractable only if documented.
- **Paced deployment (Otto-291)**: P0 substrate first (CLAUDE/AGENTS/ALIGNMENT), then memory+skills, then external surfaces. One language fully validated before second pilot begins.
- **Second pilot validates reproducibility**: two languages from different language families (e.g., Spanish + Mandarin) prove the tooling generalizes beyond the first pilot.
