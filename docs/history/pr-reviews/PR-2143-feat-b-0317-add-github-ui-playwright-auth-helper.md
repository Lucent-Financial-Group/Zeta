## Description

## Summary

- Add `tools/playwright/github-ui/auth.ts` with `withGitHubSession` for GitHub UI automation through a Playwright storage-state file.
- Validate storage-state input, fail closed on unauthenticated GitHub pages, and extract the signed-in GitHub username from page metadata.
- Add focused fake-driver tests and `.gitignore` coverage for local session-state files.
