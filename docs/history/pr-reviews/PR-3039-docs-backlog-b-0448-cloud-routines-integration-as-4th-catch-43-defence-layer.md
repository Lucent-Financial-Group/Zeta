## Why

Today the autonomous-loop substrate established three layers of catch-43 defence ([PR #3034](https://github.com/Lucent-Financial-Group/Zeta/pull/3034)): CLI session cron, Desktop routine, repo canonical. The 2026-05-13 Claude Desktop feature research surfaced a 4th Anthropic ships — **Cloud Routines** (research preview) — that adds:

- Scheduled triggers on Anthropic's infrastructure (runs even when machine is off)
- **GitHub event triggers** (\`pull_request.opened\`, \`push\`, \`issues\`, \`releases\`, \`check_runs\`) — unlock per-PR review ticks without any local cron
- API triggers (HTTP POST + bearer token) for external monitoring integrations

Usage caps: Pro 5/day, Max 15/day, Team/Enterprise 25/day.

Aaron 2026-05-13 explicitly authorized this row: *"yes that sounds good about the backlog too"* after the research synthesis.

## Composition
