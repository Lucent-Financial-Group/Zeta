  <pre>
  The quadratic Bezier loop builds segments from a point to the midpoint between points, using the previous point as the control point. This means the trajectory curve won’t actually pass through the intermediate commit points (only the first/last), so the plotted markers won’t lie on the path. Consider drawing segments that end at each `curr` point (or use a Catmull–Rom→Bezier conversion) so the spline matches the data points.
  </pre>
