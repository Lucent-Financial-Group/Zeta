## Summary

- **B-0357**: Replaces two tautological Z3 agenda lemmas (13 and 14) with genuinely non-trivial proofs that require Z3 to do real model-space search via integer linear arithmetic transitivity.
- **New Lemma 13 — Agenda monotonicity**: if `threshold_A > threshold_B`, then A's agenda ⊆ B's. UNSAT requires a 3-way inequality chain (`Quality(t) ≥ threshold_A > threshold_B > Quality(t)`); returns SAT without the ordering constraint.
- **New Lemma 14 — Agenda range disjointness**: if `hi_A < lo_B`, no trajectory qualifies for both agendas. UNSAT requires resolving `Quality(t) ≤ hi_A AND Quality(t) ≥ lo_B AND hi_A < lo_B`; returns SAT without the range constraint.
- Old tautologies preserved as `[TEACHING]` comments (shadow catch #30 teaching artifacts).
