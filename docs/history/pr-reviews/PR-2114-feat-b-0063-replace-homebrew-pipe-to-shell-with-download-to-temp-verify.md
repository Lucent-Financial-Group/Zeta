  <pre>
  P1: `mktemp` is used via command substitution (`HOMEBREW_INSTALLER_TMP="$(mktemp)"`). On macOS bash 3.2, `set -e` is not reliable in command substitutions without `inherit_errexit`, so a `mktemp` failure can silently leave an empty path and fall through to `curl_fetch --output "" ...`. Prefer an explicit guard like `if ! HOMEBREW_INSTALLER_TMP="$(mktemp)"; then ...; fi` (or `|| exit 1`) before setting the trap and downloading.

  </pre>
