## Summary

Splits B-0023 (XL quant × Austrian monetary theory synthesis) from a monolithic research blob into 8 dependency-ordered atomic backlog rows:

- **B-0380** (P3, S) — Controlled-vocabulary pass (Otto-286): precise definitions of "quant-grade rigor", "Austrian primitive", "formalization", "synthesis gap" — the root atom, gates all downstream work
- **B-0381** (P3, M) — Literature survey: catalog/classify existing partial synthesis attempts (Selgin/White, Garrison, Saifedean S2F, Steve Keen, agent-based modeling) using B-0380 vocabulary
- **B-0382** (P3, M) — Per-primitive #1: time-preference → stochastic utility-discount (Epstein-Zin SDF framework)
- **B-0383** (P3, M) — Per-primitive #2: capital-structure / Hayekian-triangle → multi-stage stochastic production (real options, DAG models)
- **B-0384** (P3, M) — Per-primitive #3: calculation-problem → information-theoretic complexity bound (distributed systems, formal complexity, Grossman-Stiglitz)
- **B-0385** (P3, M) — Per-primitive #4: ABCT → credit-cycle stochastic process with regime-switching (Hamilton 1989, Minsky comparison)
- **B-0386** (P3, M) — Per-primitive #5: sound-money → monetary-aggregate process with hard-cap constraint (commodity money models, Saifedean S2F classification, Bitcoin natural experiment)
- **B-0387** (P2, M) — Synthesis ADR: formalizability matrix + open-research publishing pipeline (capstone, integrates all five primitive verdicts)

## Design notes

The vocabulary-first dependency (B-0380) is load-bearing because the synthesis gap is itself a *definitional problem* — whether any given formalization "succeeds" depends on what "quant-grade formalization" means for an axiomatic/praxeological tradition that is methodologically distinct from empirical stochastic modeling. Without B-0380, B-0381's literature classification and B-0382–B-0386's verdicts would be based on unstated standards.

B-0384 (calculation-problem) is hypothesized to be the strongest formalization candidate since Mises's 1920 claim is already stated in computational/information terms. B-0387 either confirms or revises this hypothesis.
