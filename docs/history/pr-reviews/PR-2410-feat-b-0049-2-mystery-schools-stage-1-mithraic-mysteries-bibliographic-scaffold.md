## Shelf contents (mystery-schools-mithraic.md)

- **Overview**: Roman Mithraism (c. 70–400 CE), exclusively male, predominantly military/merchant, underground mithraea.
- **Source problem**: Almost entirely iconographic + archaeological; no initiatory scripture survives. Four source categories: archaeological record, hostile Christian apologists, Neoplatonic philosophers, the Mithras Liturgy (PGM IV.475–820).
- **Primary witnesses**: Plutarch (*Life of Pompey* 24.5 — earliest mention), Statius (*Thebaid* I.719–720), Celsus/Origen (*Contra Celsum* VI.22 — seven grades), Justin Martyr (hostile, parallel features), Tertullian (hostile, institutional detail), Porphyry (*De Antro Nympharum* — cave symbolism), Julian the Emperor (insider theological statement), Mithras Liturgy (debated Mithraic character).
- **Seven grades table**: Corax → Nymphus → Miles → Leo → Perses → Heliodromus → Pater with planetary patrons (per Origen).
- **Archaeological record**: 400+ mithraea; detailed coverage of Santa Prisca Rome, San Clemente Rome (three-layer Christian/pagan stratigraphy), Walbrook London, Carrawburgh Hadrian's Wall, Dura-Europos Syria.
- **Tauroctony catalog**: all standard elements catalogued with scholarly attributions.
- **Mithraic narrative cycle**: six-scene sequence.
- **Modern scholarship**: Cumont (1894–1903) → 1971 Manchester congress turning point → Beck, Ulansey, Clauss, Gordon, Martin, Merkelbach, Burkert.
- **Six scholarly debates table**: Iranian origin, tauroctony meaning (4 positions), Mithras Liturgy, relation to Christianity, social profile, secrecy — all without adjudication.
- **Reception history**: Renaissance, Theosophical, occult, new-age — catalogued as reception, not history.
