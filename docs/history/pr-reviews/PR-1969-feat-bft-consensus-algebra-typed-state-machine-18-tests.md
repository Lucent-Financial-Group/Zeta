  The quorum threshold is derived from `votes.Length`, which is the number of ballots received, not the configured replica set size. In a 4-node cluster, if one node is offline and only 3 votes arrive, this code computes threshold from `3` and can commit with far fewer supporters than the intended `2f+1` of the full cluster.
