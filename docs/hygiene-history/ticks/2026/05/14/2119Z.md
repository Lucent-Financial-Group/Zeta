## Notes for future-Otto

**Tick-shard authoring tip**: when a section heading is followed
immediately by a list (e.g., `## Fix` → `- item`), MD032 requires a
blank line between them. The list-after-paragraph case also triggers
MD032 — list-after-anything-non-list needs the blank line.

Pattern: write `Section:` + blank line + list. Otherwise CI catches it.
