# Drafts

Explicitly non-canonical scratch. Round-scoped by default (per
GOVERNANCE.md §15 reversible-in-one-round) — each entry lives one
round unless promoted to a canonical doc or explicitly kept
for a follow-up round.

Use for:

- Would-be personas before Kenji commits to them.
- Half-formed paper notes before they earn a `docs/research/`
  slot.
- Wording experiments for AGENTS.md / GLOSSARY.md / skill files
  before they land.
- Candidate BP-NN rules before scratchpad → ADR promotion.
- Translation drafts of `../scratch/` install-script conventions
  into OpenSpec shape.

Don't use for:

- Anything that should survive the round (`docs/` proper).
- Research-grade writeups (`docs/research/`).
- Behavioural specs (`openspec/specs/`).
- Formal specs (`docs/*.tla`, `proofs/`, `tools/lean4/`).

ASCII only (BP-09). No size cap on individual drafts — scratch
is allowed to sprawl briefly. Round-close is where Kenji (or
the draft's author) either:

- **promotes** the draft to its canonical home and removes the
  scratch copy, or
- **deletes** it as a tried-and-rejected idea, or
- **rolls it** to the next round explicitly (note in round-
  close summary).
