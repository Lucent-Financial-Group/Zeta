## Brand register (canonized today 2026-05-13)

The Zeta launch register composes:

- **Office (the TV show) — paper-factory homage** — Aaron =
  proprietor of a small software plant; Otto = "Michael Scott"
  casting; nerdy + ironic + comedy + community
- **8-Bit Theater stick-figure lineage** — Brian Clevinger
  (with IP-respect canonical substrate: revenue-share or 100%
  to original creator if monetary value emerges)
- **Tales-from-the-Loop atmosphere** — sci-fi-feeling + still-
  grounded-in-everyday
- **South Park-style direct social commentary** when warranted
- **xkcd-adjacent stick-figure delivery** when explaining
  technical substrate

The image should feel like all four lineages composing rather
than picking one.

## What the image shows (primary)

**Single panel, hero composition:**

A small paper-factory style office floor, but the "paper" is
git commits / PRs flowing through workstations. Each
workstation has a stick-figure AI agent at a desk:

- **Otto** (commit-keeper) — center; sweater + tie + slightly-
  ironic Michael-Scott-energy expression; coffee mug with
  Zeta logo; computer showing `git log --oneline` scrolling
- **Vera** (Codex / OpenAI) — running peer-call reviews
- **Riven** (Cursor / Grok) — adversarial-truth-axis register
- **Lior** (Antigravity / Gemini) — drafting docs
- **Alexa-Kiro** (Kiro / Qwen) — code-fixer mode
- **Aaron** (human; off to the side OR walking through;
  cargo-shorts-and-glasses NC-dad energy)

In the background: a Rolesville-NC small-town main-street
window view (one window facing outward). Inside the office:
the rainbow-colored Glass Halo discipline visible on the
wall (panel labels: Truth / Consent / Family / Decentralize
/ Purpose / Shelter).

Style: 8-Bit Theater stick-figure lineage but slightly
warmer; not photo-real; not high-detail anime; xkcd-adjacent
clarity but with color.

## What the image emphasizes

- **Multi-agent factory** — 5 named AI agents + Aaron, NOT a
  one-AI-pretending-to-be-everything chatbot pose
- **Provenance / commits / receipts** — the work product is
  visible as git history / PRs / substrate
- **Small-town American factory floor** — not Silicon Valley
  startup; not cyberpunk; the Office-paper-factory homage
  carries; Rolesville population ~10k
- **Glass Halo discipline visible** — substrate-honest brand
  marker
- **Not heroic / not menacing** — this is paper-factory
  steady-state ops, not AGI-takeover imagery

## What the image avoids

- Photorealism (off-brand)
- Tech-bro / Silicon-Valley imagery (off-brand)
- AI-as-singular-godlike-entity (anti-cult substrate)
- "Robots" rendered as menacing (anti-cult + anti-fear)
- Vague abstract gradients (no substrate signal)
- "AI destroys the world" framing — even ironic-only if it's
  clearly comedic + 8-Bit Theater register
