How to Read This Panel

Each clause is a candidate metric for measurable alignment. A green badge means the clause is present in docs/ALIGNMENT.md on main.


The alignment auditor agent (Sova) emits per-commit alignment signals against these clauses. Future iterations of this panel will show per-clause commit coverage, retraction counts, and a time-series alignment trajectory.
